# Brazilian National Anthem: History & Symbolism

An interactive academic project developed for the **Front-end Development** course at the **Catholic University of Brasília (UCB)**.

### Overview
This project explores the historical evolution and the classical vocabulary of the Brazilian National Anthem through a parallax-driven web experience.

### Key Sections
* **The Beginning:** Contextual history and core meanings.
* **The Lyrics:** A glossary of classical terms.
* **Timeline:** Chronological evolution (1808–1922).
